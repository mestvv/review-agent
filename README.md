## Использование

**1. Индексация PDF файлов:**
```bash
# Положи PDF файлы в папку articles/
python rag.py
```

Скрипт автоматически:
- Найдет все PDF в папке `articles/`
- Пропустит уже проиндексированные файлы (по хешу)
- Добавит новые в ChromaDB

**2. Запуск агента:**
```bash
python query.py
```

Откроется Gradio интерфейс с двумя вкладками:
- **Обзор литературы** — генерирует научный обзор по теме
- **Вопрос-Ответ** — отвечает на конкретные вопросы

**3. Программное использование:**
```python
from query import generate_review, ask

# Обзор литературы
review = generate_review("Влияние климата на вечную мерзлоту")

# Ответ на вопрос  
answer = ask("Какая скорость деградации мерзлоты?")
```

## Ключевые функции

| Файл | Функция | Описание |
|------|---------|----------|
| `rag.py` | `index_all_pdfs()` | Индексирует все PDF из articles/ |
| `rag.py` | `index_pdf(path)` | Индексирует один файл |
| `rag.py` | `clear_database()` | Очищает базу данных |
| `query.py` | `generate_review(topic)` | Генерирует обзор литературы |
| `query.py` | `ask(question)` | Отвечает на вопрос |

Система отслеживает хеши файлов, поэтому при повторном запуске `python rag.py` уже проиндексированные файлы будут пропущены — можно просто добавлять новые PDF в папку и перезапускать.
